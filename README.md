# Kalimat.ai webchat

## pre request start
- nodejs
- mongodb

## Instructions

```sh

// start mongodb 
$ sudo service mongod start

// install deps
$ npm install

// start development server
$ npm start

// akses web browser
$ http://localhost:5000/

// broadcast message to all client
$ http://localhost:5000/broadcast

```
